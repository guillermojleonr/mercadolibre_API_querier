¿Qué es Catcher?

Catcher es una aplicación desarrollada en Python-Django que hace algo muy sencillo: consultar de forma automática información relacionada con la paquetería de mercadolibre.

¿Cómo funciona?

Bastante simple:

  • Autorización: para poder acceder a los datos el data owner debe dar permisos a nuestra app para poder realizar las consultas, para ello debe acceder al menú Login y presionar el botón de registro.
  • Consultar: Querier es el módulo que permite a Catcher obtener los datos usando la API de mercadolibre, una vez registrado tendrás acceso a un formulario donde podrás escanear los códigos QR de los paquetes, realizar las consultas y obtener la información de cada envío.

Contáctanos

Catcher está orientado a transportistas que prestan el servicio de envío de última milla. Además, al usar la API de mercadolibre, podemos adaptarlo para obtener otro tipo de información relevante. Si te interesa usar Catcher, envíanos un mensaje usando el formulario de contacto de savingl.cl
